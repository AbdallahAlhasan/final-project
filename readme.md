Project Overview

This project is a web application built using Flask that allows users to:

Register with a unique username and valid email

Log in securely with hashed passwords

Access a personal dashboard and a calculator page

Log out safely

It implements user authentication, email & password validation, and a functional calculator. This demonstrates understanding of web development concepts, database management, and secure coding practices.

Features

User Registration

Unique username and email required

Password must be at least 8 characters, include uppercase, lowercase, numbers, and special characters

Securely stored using hashed passwords

User Login

Login via username or email

Session management to keep users logged in

Flash messages for errors or successful login

Dashboard & Calculator

Accessible only after login

Simple calculator implemented using HTML, CSS, and JavaScript

Error handling for invalid calculations

Logout

Clears session to log out securely

Technologies Used

Flask – Web framework

CS50 SQL – Database management

Werkzeug Security – Password hashing

HTML, CSS, JavaScript – Frontend

Jinja2 – Template rendering

Regex – Email & password validation
